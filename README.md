# Bio

Personal link-in-bio site - Swiss Modernist minimal design.

**Live:** [kartikeybhardwaj.me](https://kartikeybhardwaj.me)

## Features

- Ultra-minimal two-column layout
- Typography-driven hierarchy using Inter
- Monochromatic off-white palette
- Zero frameworks - vanilla HTML & CSS

## Tech Stack

- HTML5 / CSS3
- Google Fonts (Inter)
- Firebase Hosting

## Run Locally

```bash
git clone https://github.com/kartikeybhardwaj/site-bio.git
cd site-bio
open index.html
```
